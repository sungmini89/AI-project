# AI 감정 일기장

> AI가 분석하는 스마트한 감정 일기장 - 당신의 감정을 이해하고 성장하세요 🌟

## 🚀 주요 기능

### ✨ 핵심 기능
- **📝 리치 텍스트 에디터**: Tiptap 기반 고급 텍스트 편집
- **🧠 AI 감정 분석**: 실시간 감정 분석 (한국어/영어 지원)
- **📊 시각화 대시보드**: Chart.js 기반 감정 트렌드 분석
- **📅 감정 캘린더**: 월별 감정 히트맵
- **🏠 로컬 저장소**: IndexedDB로 완전한 프라이버시 보장

### 💡 스마트 분석
- **감정 키워드 인식**: 한국어 감정 사전 기반 분석
- **트렌드 분석**: 주간/월간 감정 패턴 파악
- **통계 제공**: 작성 습관 및 감정 분포 분석
- **인사이트 제공**: AI가 제안하는 개선점

### 🎨 사용자 경험
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화
- **다크 모드**: 눈에 편한 다크 테마
- **PWA 지원**: 앱처럼 설치 가능
- **오프라인 작동**: 인터넷 없이도 사용 가능

## 🛠️ 기술 스택

## 🧠 감정 분석 시스템

### 영어 텍스트 분석
- **Sentiment.js** 라이브러리 활용
- AFINN 사전 기반 점수 계산
- -5 ~ +5 점수 범위

### 한국어 텍스트 분석
```typescript
const koreanEmotionKeywords = {
  happy: ['행복', '기쁨', '즐거움', '신남', '좋음'],
  sad: ['슬픔', '우울', '외로움', '실망'],
  angry: ['화남', '짜증', '분노', '열받음'],
  // ... 더 많은 감정 키워드
};
```

### 지원 감정 타입
- 😊 happy (행복)
- 😢 sad (슬픔) 
- 😠 angry (화남)
- 😐 neutral (중립)
- 🎉 excited (신남)
- 😌 calm (차분함)
- 😰 anxious (불안)
- 😤 proud (자랑스러움)
- 😞 disappointed (실망)
- 🙏 grateful (감사)

## 🔒 개인정보 보호

### 완전한 로컬 저장
- 모든 일기 데이터는 브라우저에만 저장
- 외부 서버로 전송되지 않음
- 사용자만 데이터에 접근 가능

### 데이터 관리
- **백업/복원**: JSON 파일로 데이터 내보내기/가져오기
- **자동 백업**: 정기적 로컬 백업
- **데이터 삭제**: 완전한 데이터 제어

## 🎯 사용법

### 1. 일기 작성
1. **새 일기 작성** 버튼 클릭
2. 제목과 내용 입력
3. 태그 추가 (선택사항)
4. 실시간 감정 분석 확인
5. 자동 저장 또는 수동 저장

### 2. 감정 분석 확인
1. **분석** 페이지 접속
2. 기간별 감정 트렌드 확인
3. 감정 분포 차트 분석
4. AI 인사이트 확인

### 3. 일기 관리
1. **일기 목록** 에서 과거 일기 검색
2. 감정별, 날짜별 필터링
3. 일기 수정 및 삭제
4. 태그 기반 분류
